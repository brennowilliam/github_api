# Github API Frontend

## How to start
1. Clone the repo: `git clone github_name`
2. `cd repo_name/client`
3. `npm install`
4. `npm start`

## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
```
